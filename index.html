1. 公式 α = F / [π(D₁ + D₂)] 中，为什么分母使用 π(D₁ + D₂) 而非单纯的环周长？

✗ 表面张力定义为单位长度上的力，公式中分母必须严格对应实际受力线长度，绝非历史习惯。

✓ 正确！拉脱法中液体同时润湿金属环的内外两侧，接触线总长 = πD₁(外) + πD₂(内)。

✗ D₁、D₂ 分别代表金属环的外径和内径（单位 mm），相加求总周长（非相乘求面积），接触线与"液膜面积"无关。

✗ 公式源自拉脱法基本原理：α = F / L，其中 L 为液膜与环的实际接触线总长度，有严格的物理推导，非经验凑数。

2. 实验时为什么要保证金属环严格水平且洁净（完全润湿）？

✗ 环的水平度和洁净度直接决定润湿均匀性，是实验成败的关键前提。水平偏差或污渍会导致有效接触线产生不可预估的偏离。

✓ 正确！润湿不均匀意味着部分区域液体未附着或附着异常，有效接触线不再是完整的 π(D₁+D₂)，引入显著系统误差。

✗ 环的自重可通过传感器调零消除，与水平度无关。倾斜的环主要影响的是液体润湿的均匀性而非自重。

✗ 传感器防护与环的水平度和洁净度完全无关。这是测量精度问题，不是设备保护问题。

3. 若操作者在液膜尚未完全拉断时就匆忙读取了电压值，最终计算的表面张力系数会如何？

✗ 拉脱法要求读取液膜断裂瞬间的最大值 U_max。若未到断裂点，电压仍处于上升阶段，读数值小于真正的 U_max。

✓ 正确！U_max 出现在液膜恰好断裂的瞬间（C点），在此之前电压还在上升，读数必然偏小。由 α ∝ U_max 知，α 也偏小。

✗ 液膜拉断前后电压有显著差异——拉断前电压持续上升到峰值，拉断瞬间垂直骤降，所以"无差别"是错误的。

✗ 结果是确定的，可从实验原理严格推导：提前读数 → 电压偏小 → α 偏小。不存在"无法确定"的情况。
